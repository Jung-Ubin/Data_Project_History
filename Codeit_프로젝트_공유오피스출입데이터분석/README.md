# 공유 오피스 출입 데이터 기반 결제 전환 분석

Codeit 데이터 분석 프로젝트 (팀 프로젝트)
프로젝트 기간: 25.09.03 ~ 25.09.15

---


## 프로젝트 배경
공유 오피스 서비스는 무료 체험 이후 유료 전환율이 낮고,  
체험 데이터를 어떻게 비즈니스 전략으로 연결할지에 대한 고민이 필요했습니다.  

본 프로젝트는 공유 오피스의 출입 데이터를 기반으로 

- 체험 이용 패턴과 결제 전환의 관곅를 분석하고
- 내부 데이터의 한계를 보완하기 위해 **외부 데이터(J사 사례, 지점 주변 사무직 밀집도 조사)**를 결합하여
  **현실적인 수익 개선 전략**을 도출하는 것을 목표로 진행했습니다.

---


## 분석 목표
- 무료 체험 이용 패턴이 결제 전환에 미치는 핵심 요인 파악
- 결제 / 미결제 고객의 행동 차이 분석
- 고객군 세분화 및 전환 가능성이 높은 타겟 그룹 정의
- 데이터 기반 서비스·가격·입지 전략 제안

---

## 분석 방법

- 데이터 전처리 및 유저 단위 속성 테이블 생성
- 다변량 로지스틱 회구 분석 (결제 요인 해석)
- XGBoost 모델을 통한 중요 변수 탐색
- 체험 일차·체류시간 패턴 EDA
- PCA + K-Means 기반 고객군 클러스터링
- 외부 데이터 결합을 통한 지점 입지 분석

--- 

## 주요 분석 결과

### 1. 결제 전환 핵심 요인
- **체험 일차**가 늘어날수록 결제 확률 증가
- 체류 시간은 결제율과 음의 상관관계를 가짐  
  → 장시간 체류 = 서비스 불만족이 아닌 타 공유 오피스 대비 비싼 가격 부담 가능성
- 18 ~ 20시 입실 고객의 결제율이 가장 높음

### 2. 고객군 세분화(K-Means)
- 단기 체류 + 사무직 밀집 지역 지점 이용 고객군의 결제율이 가장 높음
- 비사무 밀집 지역에서 장기 체류하는 고객군은 결제율이 가장 낮음
- 지점 입지와 고객 행동 패턴이 결제율에 유의미한 영향

### 3. 무료 체험 문제점
- 무료 체험을 3일 모두 이용한 고객은 23%
- 예약일을 넘겨 방문한 고객이 77%
- 체험 1일만 이용하고 이탈하는 고객 다수 존재

## 개선 전략 제안

### 1. 회원권 상품 다양화
- 18시 ~ 20시 이외에 이용한 고객에 특화된 할인된 회원권 제공 제안
- 18 ~ 06시 까지 이용가능한 할인된 회원권 제공 제안

### 2. 첫 체험 경험 개선
- 체험 1일차 고객 대상 맞춤 가이드 제공
- 체험 이용 후 간단 설문을 통한 초기 이탈 원인 수집
  → 조기 이탈 감소 및 전환율 상승 기대

### 3. 입지 전략 고도화
- 사무직 밀집도가 높은 지역 중심 지점 확장 제안
  → 결제율 및 재방문율 동시 개선

### 4. 무료 체험 시스템 개선
- 무료 체험 시작일 변경 기능 제공
- 체험 시작 전 리마인드 메시지 발송
- 날짜별 예약 인원 확인 UI 제안
  → 체험 만족도 및 참여율 개선

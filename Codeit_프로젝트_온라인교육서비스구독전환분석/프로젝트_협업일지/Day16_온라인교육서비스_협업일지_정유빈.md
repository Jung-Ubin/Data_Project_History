## \* 사용했던 코드

### 데이터 셋 오류 발견 (첫 결제 완료 시간보다, 구독 종료버튼 클릭 시간이 더 빠른 유저들은 몇 명있는지(이상치) 파악)

```python
# 문자열 → datetime 형 변환
user_df['첫 결제 완료 시간'] = pd.to_datetime(user_df['첫 결제 완료 시간'])
user_df['구독 종료 버튼 클릭 시간'] = pd.to_datetime(user_df['구독 종료 버튼 클릭 시간'])

# 조건에 맞는 유저 필터링
invalid_cancel_users = user_df[
    user_df['구독 종료 버튼 클릭 시간'] < user_df['첫 결제 완료 시간']
]

# 몇 명인지 출력
print("이상치 유저 수:", len(invalid_cancel_users))

# 해당 유저들의 정보 확인
invalid_cancel_users.head()
```

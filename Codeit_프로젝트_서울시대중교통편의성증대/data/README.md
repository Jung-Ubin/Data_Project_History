# 데이터 설명

본 디렉토리는 서울시 대중교통 편의성 분석에 사용된  
원본 데이터 및 전처리된 데이터를 정리한 공간입니다.

---

## 데이터 출처
- 서울열린데이터광장
- 서울시 교통정보 시스템(TOPIS)
- 국토교통부 V-World

---

## 데이터 구성
- `raw/` : 수집한 원본 데이터
- `processed/` : 분석을 위해 전처리한 데이터
- `derived/` : 분석 과정에서 새로 생성한 데이터

--- 

## 참고사항
- 일부 데이터는 분석 목적에 맞게 컬럼 선택 및 필터링이 적용되었습니다.
- 분석에 필요한 지표가 존재하지 않는 경우,
  좌표 기반 계산을 통해 새로운 변수를 생성했습니다.
